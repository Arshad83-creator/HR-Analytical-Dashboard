##Project Objective (Main Objective)

To analyze employee data using Power BI in order to identify attrition patterns, workforce distribution, and key HR metrics that help management improve employee retention and decision-making.

## Data set used
- <a href="https://github.com/Arshad83-creator/HR-Analytical-Dashboard/blob/main/HR%20Data.xlsx">Dataset</a>

## Question(KPIs)

-What is the total number of employees?
-How many employees left the company?
-What is the attrition rate shown in the dashboard?
-How many active employees are there?
-What is the average age of employees?
-Which department has the highest attrition?
-Which age group has the most employees?
-Why is attrition rate an important KPI?
-What does job satisfaction show?
-How does this dashboard help HR?
